# Data Model: Deferred Editor Overlays

## Deferred Debug Scene Renderer

Editor renderer used by Scene View.

Attributes:
- Deferred main scene path inherited from `DeferredSceneRenderer`
- Editor pass descriptors registered by name
- Optional debug scene descriptor containing selected/highlighted actors and picking request

Relationships:
- Consumes `BaseSceneRenderer::SceneDescriptor`
- Produces `RenderScenePackage` through deferred package construction
- Owns editor helper pass scheduling

## Deferred Package Extension

Generic runtime structure/function flow used to append additional pass metadata and command inputs after deferred scene passes.

Attributes:
- Base deferred metadata: GBuffer and Lighting
- Extension metadata: caller-provided helper/auxiliary passes
- Base deferred pass inputs
- Extension pass inputs

Relationships:
- Used by `DeferredSceneRenderer` unchanged for normal deferred rendering
- Used by `DebugSceneRenderer` to append editor overlays

## Editor Overlay Pass Input

Prepared `RenderPassCommandInput` generated by editor passes.

Attributes:
- Command kind, usually Helper
- Optional recorded draw commands
- Optional color/depth attachments for picking
- Draw count and readback-relevant texture views

Relationships:
- Appended after deferred lighting
- Picking input may register a preferred readback texture
